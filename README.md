# CRUD SPRING BOOT

### Instale o docker no seu computador:

https://docs.docker.com/desktop/

### Clone o projeto:
```bash
git clone https://github.com/Pedrocardo005/crudspringboot
```

### Rode o projeto
```bash
./run.sh
```

ou se preferir rode o comando:
```bash
docker-compose -f docker-compose.yml up -d
```
